# Политика конфиденциальности — Verona Universal Timer

Дата вступления в силу: 2026-04-29

Verona Universal Timer (далее — «Приложение») — это таймер сна для Android TV.
В этом документе описано, какие данные обрабатываются Приложением и для каких целей.

## 1. Сбор персональных данных

Приложение **не собирает**, **не хранит** и **не передает** персональные данные пользователя разработчику или третьим лицам.

Для использования Приложения не требуется регистрация аккаунта.
Приложение не использует рекламные SDK, аналитические трекеры и системы профилирования.

## 2. Разрешения устройства

Для работы функций таймера на некоторых моделях Android TV Приложение может запрашивать:

- **SYSTEM_ALERT_WINDOW** («Поверх других приложений»)  
  Используется только для отображения экрана сна в момент срабатывания таймера на устройствах, где это требуется прошивкой.

- **VIBRATE**  
  Используется для локального поведения интерфейса на поддерживаемых устройствах.

Эти разрешения используются только на устройстве и не применяются для отслеживания пользователя.

## 3. Локальное хранение данных

Приложение сохраняет локально на устройстве только технические данные, необходимые для работы таймера (например, время срабатывания).
Эти данные не отправляются на серверы разработчика и не передаются третьим лицам.

## 4. Сторонние сервисы

Приложение не передает персональные данные сторонним сервисам.
При установке и распространении через магазины приложений (Google Play, RuStore) соответствующие платформы могут обрабатывать технические данные в рамках своих собственных политик конфиденциальности.

## 5. Дети

Приложение не ориентировано специально на детей и не осуществляет намеренный сбор персональной информации о детях.

## 6. Безопасность

Поскольку Приложение не собирает персональные данные на удаленные серверы, у разработчика отсутствует централизованная база персональных данных пользователей данного Приложения.

## 7. Изменения политики

Разработчик может обновлять настоящую Политику конфиденциальности.
Актуальная версия всегда публикуется по тому же URL с обновленной датой вступления в силу.
