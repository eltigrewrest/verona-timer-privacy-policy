# Privacy Policy — Verona Universal Timer

Effective date: 2026-04-29

Verona Universal Timer (“the App”) is a TV sleep timer application.
This Privacy Policy explains what data the App collects and how it is used.

## 1. Data Collection

The App does **not** collect, store, or transmit personal data to the developer or third parties.

The App does not require account registration.
The App does not use analytics SDKs, advertising SDKs, or trackers.

## 2. Device Permissions

The App may request the following permissions strictly for timer functionality on Android TV devices:

- **SYSTEM_ALERT_WINDOW** (“Display over other apps”)  
  Used only to show the sleep overlay screen when the timer triggers, on devices where this behavior is required by firmware.

- **VIBRATE**  
  Used for device-side interaction behavior where supported by the system.

These permissions are used locally on the device and are not used for tracking.

## 3. Local Storage

The App stores timer state (for example, scheduled trigger time) locally on the device using standard Android local storage mechanisms.
This data is not shared externally.

## 4. Third-Party Services

The App does not send personal data to third-party services.
If the app is distributed via app stores (Google Play, RuStore), those platforms may process technical data under their own privacy policies.

## 5. Children’s Privacy

The App is not directed to children specifically and does not knowingly collect personal information from children.

## 6. Security

Since the App does not collect personal data to remote servers, there is no user personal database maintained by the developer for this App.

## 7. Changes to This Policy

This Privacy Policy may be updated from time to time.
The updated version will be published at the same URL with a new effective date.
